# 4-DOF Robotic Arm
## Overview
This 4-degree-of-freedom robotic arm is currently in progress of being built. The arm is built utilizing 4 MG90 micro servos which will be controlled with 4 10k Ohm Potentiometers and an ESP32 microcontroller.
